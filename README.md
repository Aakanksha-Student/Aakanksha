# PowerBI_Dashboard
1.  Project Title
FestMan Stores ‚Äì Financial Performance Analysis Dashboard

2. Short Project Description

This project presents an interactive Financial Analysis Dashboard built to analyze the sales, profit, orders, discounts, and profit margins of FestMan Stores across multiple countries, segments, and products. The dashboard helps stakeholders track year-over-year performance, identify high-performing regions and segments, and understand the impact of discounts on profitability for data-driven decision-making.

3. Tech Stack

Power BI- Data visualization & dashboard creation
MS Excel / CSV - Data cleaning and preparation
DAX - Measures and calculated KPIs
Power Query - Data transformation

4. Data Source

Sample retail sales dataset (CSV/Excel format)
Includes data on sales, orders, profit, discount bands, countries, products, segments, and time (year & month).

5. Features & Highlights

a.Key Performance Indicators (KPIs):

Sales, Orders, Profit, Profit Margin, Discount (Current vs Prior Year).

b.Geographical Analysis:

   Orders and Profit Margin by Country

c.Segment Analysis:

Profit Margin by Business Segment (Channel Partners, Enterprise, Government, etc.)

d.Product Performance:

Top Products by Sales
Profit Margin by Product & Segment
    
e.Discount Analysis:

Distribution of discounts across Low, Medium, and High bands

f.Time-Series Insights:

Monthly and yearly sales trend analysis

g.Interactive Dashboard:

Filters and visuals for quick insights and comparisons

6. Key Insight:
Channel Partners show the highest profit margin, while Enterprise segment indicates negative profitability, highlighting areas for strategic improvement.
